План лікування Профілактика Онко-вікі Туморборд

Оцінка ризику для профілактики / ранньої діагностики

Інструмент для лікаря (HCP-mediated). Заповніть профіль пацієнта з ризику нижче. Система ідентифікує prevention-eligible фактори ризику (інфекційні канцерогени IARC Group 1, червоні прапори спадкового родоводу, професійні впливи, хронічні стани, ятрогенні + lifestyle) і генерує структурований Prevention Plan з двома треками (стандартна інтервенція + альтернатива зі спостереженням) за CHARTER §15.2 C4. Це не медичний пристрій; завжди звіряйтесь з лікарем-куратором.

A. Демографія пацієнта
B. Хронічні інфекції (IARC Group 1)

Кожна підвищує ризик специфічних раків; інтервенція — curative або risk-reducing противірусна / антибіотична / вакцина-терапія.

C. Сімейний родовід (червоні прапори спадкового раку)

Обчислені прапори. Pedigree-to-criteria evaluator — v0.2-C розширення engine (наразі очікує флаги попередньо обчислені клініцистом).

D. Lifestyle & екологічні фактори ризику
E. Професійні експозиції (IARC Group 1)
F. Хронічні медичні стани
G. Ятрогенні фактори ризику (попередні лікування)

Prevention RedFlags, які спрацюють:

    Кожен спрацьований RF маршрутизує до 2-track Prevention Plan (інтервенція + спостереження за CHARTER §15.2 C4). Натисніть Відкрити у Plan Builder, щоб запустити engine і побачити повний вивід.

    CHARTER §15 C1 інваріант: OpenOnco — HCP-mediated інструмент, не patient-direct медичний пристрій. Ця анкета — для prevention-консультацій під керівництвом лікаря. Весь контент — STUB, очікує two-Clinical-Co-Lead signoff за §6.1 dev-mode exemption.

    Дані: нічого тут не передається на сервер. Усе обробляється client-side після натискання Відкрити у Plan Builder (який вбудовує OpenOnco engine через Pyodide).

    Покриття: 50+ prevention-пілотів через 7 категорій факторів ризику (інфекційні / спадкові / хронічні стани / професійні / ятрогенні / lifestyle / репродуктивні) за IARC Monographs + USPSTF + NCCN Genetic/Familial High-Risk + AGA + AASLD.
